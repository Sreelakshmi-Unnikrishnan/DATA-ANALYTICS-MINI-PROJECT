This project analyzes customer purchase behavior using a structured dataset and an interactive Power BI dashboard.
The goal is to uncover insights into customer demographics, buying patterns, product preferences, seasonal trends, and promotional impacts.
These insights help businesses improve marketing strategies, boost sales, and optimize customer engagement.

Objectives

1.Understand the demographic profile of customers (Age, Gender, Location).

2.Analyze purchasing behavior such as:
  Item purchased
  Category
  Size & color preferences
  Seasonality
  Review ratings

3.Evaluate the impact of discounts, promotions & subscription status.

4.Identify preferred payment and shipping methods.

5.Build an interactive Power BI dashboard for dynamic exploration.
